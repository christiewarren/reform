Restrictions Breed Creativity

While we’ve all received the same education, each of us takes a completely different approach when handed a problem. This poster series highlights these individual thought processes; it illustrates how we each can start with a similar base but end with completely different outcomes.

Poster Installation

The installation of our poster series can currently be seen behind the State Theatre on Calder Way in downtown State College. We printed, cut and installed the posters by hand. Special thanks to the State Theatre for allowing us to showcase our work here!
